# Google Scholar 学术搜索技能

当你需要使用Google Scholar搜索学术文献时，使用此技能。该技能能够根据关键字搜索学术文献，并返回结构化的文献信息，包括标题、作者、摘要、引用次数、发表年份、期刊名称和文献链接。

## 使用说明

1. **搜索关键字处理**：
   - 接受自然语言查询或关键词组合
   - 支持多关键字搜索（用空格分隔）
   - 可以包含领域、主题、技术名称等

2. **结果筛选和排序**：
   - 使用 `top_n` 参数控制返回结果数量（默认10条）
   - 使用 `sort_by` 参数选择排序方式：
     - `"relevance"`：按相关度排序（默认）
     - `"date"`：按日期排序，获取最新研究
   - 使用 `start_year` 和 `end_year` 筛选特定年份范围的文献

3. **结果处理**：
   - 返回的JSON包含 `status`、`results_count` 和 `results` 字段
   - 每个结果包含：title、authors、publication、year、abstract、citations、url
   - 部分文献可能缺少某些字段（如摘要），这是正常的

4. **错误处理**：
   - 检查返回的 `status` 字段
   - 如果 `status` 为 `"error"`，查看 `message` 字段了解错误原因
   - 常见错误：关键字为空、网络连接失败、请求被限制

## Examples

- **基础学术搜索**：
  ```
  搜索关键字："machine learning"，返回前10条最相关的结果
  ```

- **获取最新研究**：
  ```
  搜索关键字："quantum computing"，按日期排序，返回2020年以来的最新研究，限制5条结果
  ```

- **特定领域文献调研**：
  ```
  搜索关键字："climate change adaptation"，筛选2022-2025年发表的文献，返回前8条结果
  ```

## Guidelines

- **合理使用频率**：避免过于频繁的请求，可能导致IP被Google Scholar暂时封禁
- **遵守使用条款**：仅用于学术研究目的，不要进行大规模商业数据抓取
- **结果验证**：部分文献信息可能不完整，建议结合文献链接进行验证
- **关键字优化**：使用准确的关键字和术语能获得更好的搜索结果
- **年份筛选**：当需要特定时期的研究时，使用年份范围参数提高结果相关性
- **引用次数参考**：引用次数可以作为文献影响力的参考，但不是唯一标准
- **多角度搜索**：对于复杂主题，可以尝试不同的关键字组合以获得更全面的结果
